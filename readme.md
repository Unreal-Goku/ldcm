## 📋 What is ldcm?

ldcm stands for Large Depth Completion Model. It fills in missing depth information using sparse measurements. This means the software takes a few depth points, like from a sensor, and creates a full depth map. This can help improve 3D models and images. ldcm is designed to work with Windows and requires no programming knowledge to use.

This tool is based on recent research presented at ICLR 2026. It is suited for users working with 3D data, robotics, or any task needing detailed depth information from limited input.
